## Team Turco – High Performance Sports Agency

Team Turco is a high-performance sports agency focused on developing athletes physically, mentally, and strategically.
Built with a modern front-end approach, this project showcases a dynamic and scalable website featuring athlete profiles, trainers, performance-focused content, automatic contact page and translation

## 📌 Project Overview

This is a real company project was created as part of a front-end development journey, combining:

- Dynamic content rendering
- Local Array fetching
- User experience (UX) improvements
- Performance-focused design

The goal is to provide a real digital platform for a global sports agency to cast and sell
